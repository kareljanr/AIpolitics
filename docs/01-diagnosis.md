# 01 — Diagnosis: the Belgian spiral

**Purpose:** state the problem without romanticism, and **tag every major claim** by strength.  
Update this file when better data arrives. Do not let rage outrun evidence.

## Thesis

Belgium is in a **slow fiscal and incentive spiral**:

1. Extremely high taxes on productive labour  
2. High public spending and structural deficits  
3. Complex subsidies and regulations that reward clients, not value  
4. A welfare + migration + voting interaction that can amplify non-contribution  
5. A multiparty system that punishes fiscal honesty and rewards spending coalitions  

If unchanged long enough, the end states look less like Denmark and more like **high-tax stagnation + fragmentation** (Latin American fiscal patterns, Lebanese clientelism, or low-trust identity politics — as *risk analogies*, not identity insults).

## Claim ledger

### C1 — Belgium taxes labour at world-class pain levels

| | |
|--|--|
| **Claim** | For average single workers, Belgium has the highest (or among the highest) OECD labour tax wedges (~52–53% of labour cost). |
| **Confidence** | **Strong** |
| **Mechanism** | Personal income tax + employee/employer social contributions inflate the cost of hiring and reduce take-home pay. |
| **Source class** | OECD *Taxing Wages* (2025: ~52.6%; 2026 brochure: Belgium 52.5% highest among listed OECD cases). |
| **Falsifier** | Sustained OECD ranking outside top 3 for comparable household types *and* falling effective wedge on hiring. |
| **Implication** | Any growth strategy that ignores the wedge is cosplay. |

Also true (Strong): total tax system includes VAT, property-related taxes, and sectoral levies. “Highest taxes in the world on everything” is **too broad** — specify labour vs consumption vs capital.

### C2 — Structural deficit + high debt

| | |
|--|--|
| **Claim** | General government runs large structural deficits; debt/GDP is above 100% and rising under current trajectories. |
| **Confidence** | **Strong** (levels); **Medium–Strong** (path without reform) |
| **Source class** | National Bank of Belgium EDP notifications; European Commission forecasts. Example magnitudes: deficit ~4.4% GDP (2024) → ~5.2% (2025) in NBB release; debt ~104–108% range in recent notifications/forecasts. |
| **Falsifier** | Multi-year primary surplus path that stabilises debt without one-off accounting tricks. |
| **Implication** | “We can keep all subsidies and cut taxes” is fantasy. Choices are mandatory. |

### C3 — Spending is high and sticky

| | |
|--|--|
| **Claim** | Public expenditure as % of GDP is high vs OECD average; primary spending is sticky (pensions, health, social, interest). |
| **Confidence** | **Strong** on level; **Strong** on stickiness of entitlements |
| **Source class** | OECD *Government at a Glance*; NBB public finance tables (primary expenditure ~52% GDP order of magnitude in recent years). |
| **Implication** | DOGE-style cuts must hit **programmes**, not only “administration” slogans. |

### C4 — The system incentives non-work and complexity

| | |
|--|--|
| **Claim** | Marginal effective tax rates + benefits phase-outs + bureaucracy often make low-productivity work unattractive; complexity rewards insiders and consultants. |
| **Confidence** | **Medium–Strong** (well-documented in EU labour literature; Belgium-specific traps need ongoing measurement) |
| **Mechanism** | High replacement rates, traps at benefit cliffs, black economy, administrative jobs feeding administrative jobs. |
| **Falsifier** | Low inactivity, low long-term benefit dependency, short benefit-to-work transitions across skill levels. |
| **Policy response** | Flatten traps; condition support; simplify. See programme. |

### C5 — Migration + welfare + voting can form a vicious circle

| | |
|--|--|
| **Claim** | If inflows are skewed toward low net fiscal contribution **and** parties compete by expanding transfers to those groups **and** voting power follows, the median voter may drift against fiscal sustainability. |
| **Confidence** | **Medium** as general political-economy model; **case-by-case** for Belgium cohorts (needs open data, not slogans). |
| **What is NOT claimed** | “All migrants are lazy” — false and stupid. Many migrants are net contributors. Selection and rules matter. |
| **What IS claimed** | Open-ended low-skill inflows into high-transfer systems without assimilation requirements is **not free**. Europe has multiple case studies. |
| **Falsifier** | Cohort data showing persistent positive net contribution + high employment for recent non-EU inflows under current rules. |
| **Policy response** | Contribution-oriented migration, language/work conditions, anti-fraud, end of parallel societies — *without* collective ethnic guilt. |

### C6 — Pensions and demographics are a structural bomb

| | |
|--|--|
| **Claim** | Ageing + early exit cultures + indexation mechanics stress public finances for decades. |
| **Confidence** | **Strong** on direction; **Medium** on exact euro gap year-by-year |
| **Source class** | Ageing reports (EU/Belgian Study Committee on Ageing); NBB/Federaal Planbureau. |
| **Implication** | Honest parties talk retirement age, career length, and second/third pillar — not only “respect for seniors” posters. |

### C7 — Subsidy state wastes large sums on low-value or absurd programmes

| | |
|--|--|
| **Claim** | Belgium (federal + regions + local) runs a thicket of subsidies, tax expenditures, and quasi-public bodies with poor cost-benefit. |
| **Confidence** | **Strong** that waste exists; **Weak–Medium** on any single viral “€X billion absurd” number until itemised |
| **Response** | Build the [DOGE Belgium leaderboard](06-doge-belgium.md) item by item. No vibes-only totals. |

### C8 — The party system rewards fiscal irresponsibility

| | |
|--|--|
| **Claim** | PR multiparty coalitions, linguistic split, and public party finance create weak incentives for painful reform and strong incentives for clientelism. |
| **Confidence** | **Medium–Strong** (comparative politics + Belgian experience of long formations and soft budgets) |
| **Implication** | A new party must either **force agenda with blackmail power** (kingmaker) or **own a reform brand** so strong that partners must buy it. See strategy & coalitions. |

### C9 — “AI alone would govern better”

| | |
|--|--|
| **Claim** | Replacing politicians with AI would automatically produce better outcomes. |
| **Confidence** | **Weak / Speculative** as governance design; **Strong** as *marketing hook* |
| **Reality** | AI has no legitimacy, no monopoly on force, no legal personality for office. Humans remain accountable. AI is for analysis, drafting, simulation, transparency. |
| **Use** | “AI-augmented truth party” — yes. “Robot PM” — no (legal fiction + constitutional dead end). |

### C10 — Without change, Belgium-as-we-know-it erodes

| | |
|--|--|
| **Claim** | Continued high wedge + deficit + low growth + social fragmentation leads to lower living standards and higher identity conflict. |
| **Confidence** | **Medium** (trend risk, not calendar prophecy) |
| **Not claimed** | Specific year of collapse. Serious projects avoid apocalypse cosplay while treating risk as real. |

## Root cause model (simplified)

```text
High labour taxes + complex subsidies
        ↓
Weaker formal employment / black economy / talent exit
        ↓
Narrower tax base + higher social spending need
        ↓
Higher taxes or higher deficit (often both)
        ↓
Political competition for transfer voters + protected insiders
        ↓
Harder reform → repeat
```

Break the loop at **taxes on work**, **spending quality**, **migration selection**, and **political incentives for honesty**.

## What we refuse in the diagnosis

- Blaming “the French” or “the Flemish” as genetic destiny  
- Pretending Wallonia/Flanders fiscal gaps are imaginary *or* that they justify every grievance  
- US culture-war imports with zero Belgian mechanism  
- “All politicians are retards” as analysis (selection effects and incentives explain more)
