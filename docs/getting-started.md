# Getting started

How to go from this repository to a credible local political movement.

## Phase 0 — Core group (now)

**Goal:** 5–15 people who share the vision and will meet monthly.

- [ ] Read [vision](vision.md) and [programme](programme.md); mark disagreements in issues
- [ ] Identify skills needed: law/electoral, comms, finance, policy, grassroots
- [ ] Set up a Signal or Matrix group (avoid Facebook-only)
- [ ] First meetup: café in Gent (Patershol, Dok Noord, or Watersportbaan area)

**You do not need a party name yet.**

## Phase 1 — Programme & presence (3–6 months)

- [ ] Consolidate programme into a 2-page **Gent manifesto**
- [ ] Choose name ([naming ideas](naming.md))
- [ ] Simple website (GitHub Pages or similar)
- [ ] Register social accounts
- [ ] Start a mailing list (GDPR-compliant)

## Phase 2 — Legal & electoral (Belgium-specific)

Belgian party law is complex. Get professional advice early.

### Options

| Structure | Use case |
|-----------|----------|
| **Informal movement** | Advocacy, no candidates |
| **VZW / non-profit** | Events, research, fundraising limits |
| **Political party (partij)** | Electoral lists, subsidies — requires statutes, minimum members, registration |

### Ghent municipal elections (October 2026)

- Electoral list requires **candidates + supporting signatures** (check current Flemish/Ghent rules)
- Consider **cartel** with like-minded liberals if standalone list is too heavy
- Budget: deposits, printing, campaigns — plan €15k–50k minimum for serious city-wide run

### Key contacts to research

- [Ghent City Council](https://stad.gent) — electoral services
- [Flemish Interior](https://www.vlaanderen.be) — party registration
- [FPS Interior](https://www.ibz.be) — federal party law if scaling up

## Phase 3 — First campaign

- Ward-level listening tours (Sint-Amandsberg, Mariakerke, Wondelgem, centrum)
- Publish **scorecard** on city council votes (competence narrative)
- Target **young professionals and entrepreneurs** — they're underserved
- Earned media: op-eds in De Tijd, Het Laatste Nieuws Gent, local podcasts

## What success looks like

| Milestone | Minimum | Ambitious |
|-----------|---------|-----------|
| Municipal 2026 | 1–2 seats, voice in coalition talks | 3+ seats, policy wins |
| Name recognition | 5% in Gent polls | 10%+ |
| Membership | 100 active | 500+ |

## Red lines

- No opaque funding
- No personal cult — rotate spokespeople
- No alliances that contradict core economic liberalism for short-term seats
