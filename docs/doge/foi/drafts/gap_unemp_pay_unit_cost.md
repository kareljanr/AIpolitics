### 1. Voorwerp

1. Voor de jaren **2022–2025**:
   - aantal dossiers / uitkeringsgerechtigden betaald via **elke**

   RVA/ONEM of Rekenhof die berekent.

4. Juridische basis van de toekenning van publieke middelen aan private
   uitbetalingsinstellingen voor deze taak.

### 2. Context

Werkloosheidsverzekering is een kerntaak van de sociale zekerheid.
Betaling via meerdere kassa’s (vakbonden + hulpkas) vraagt om
kostentransparantie t.o.v. één digitaal publiek betaalkanaal.
