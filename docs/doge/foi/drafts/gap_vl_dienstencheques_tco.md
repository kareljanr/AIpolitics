# FOI draft — gap_vl_dienstencheques_tco

**Status:** draft  
**gap_id:** gap_vl_dienstencheques_tco  

## Voorwerp

1. Totale **publieke kost** van het Vlaamse **dienstenchequesysteem** voor de laatste twee afgesloten begrotingsjaren (subsidie aan uitgevers/bedrijven, administratie, eventuele andere posten) — in euro.
2. Aantal cheques / uren en begrotingscodes.
3. Raming 2025–2026 na prijsaanpassingen en afbouw fiscale aftrek.

## Waarom

Centenboekje noemt slechts een **inspanning/besparingslijn** (€151,4 m), niet de volledige TCO.
