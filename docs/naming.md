# Party name ideas

Working names for discussion. Criteria: memorable in Dutch, not cringe in English, available domains/handles where possible.

## Shortlist

| Name | Rationale | Risk |
|------|-----------|------|
| **Vrij Gent** | Direct, local, liberal ("vrij" = free) | Generic |
| **Ratio** | Smart, evidence, calm | Hard to own as brand |
| **Volt Gent** | — | Taken / confusion with Volt Europa |
| **Liberaal Gent** | Clear positioning | Sounds like a committee |
| **Gent Werkt** | Outcome-focused, positive | Could sound labour-party |
| **Nieuw Liberaal** | Break from Open VLD baggage | Needs strong visual identity |
| **De Factoren** | Ghent industrial heritage + "factors that matter" | Obscure without explanation |
| **Slim Stad** | "Smart city" literally | Too municipal-tech |

## Naming principles

1. **Say liberal without hiding** — voters should know what they're getting
2. **Ghent-first** — we're not launching a federal party on day one
3. **Avoid empty verbs** — "Vooruit", "Verander", "Samen" are exhausted
4. **Check**: [KBO/BCE search](https://kbopub.economie.fgov.be/kbopub/zoeknaamfonetischform.html), `.be` domain, Instagram/X handle, electoral list rules

## Tagline options

- *Vrij. Slim. Gent.*
- *Economisch liberaal. Lokaal concreet.*
- *Minder praten. Meer bouwen.*
- *Competentie is ook progressief.*

## Visual direction (informal)

- Clean typography, no flags-waving nationalism
- Ghent colours reimagined (not copying city marketing)
- Avoid orange (Open VLD) and purple (Volt) as dominant — differentiate

## Decision process

Run a **member poll** after 20+ committed supporters. Until then, repo stays **Politics**.
