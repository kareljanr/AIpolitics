# Claim tags (quick reference)

| Tag | Use when |
|-----|----------|
| **Strong** | Multiple solid sources or accounting identity (e.g. OECD tax wedge rank, NBB deficit notification) |
| **Medium** | Good theory + partial data; Belgium-specific magnitude still fuzzy |
| **Weak** | Plausible single source or analogy |
| **Speculative** | Scenario / story — do not legislate on this alone |

## Source classes (examples)

- OECD *Taxing Wages*  
- National Bank of Belgium (EDP / public finance)  
- European Commission forecasts  
- Court of Audit (Rekenhof)  
- FPS / FOD official stats  
- Regional budgets (Flanders, etc.)  
- Peer-reviewed labour economics  
- FOI / parliamentary questions  

## Phrases to avoid without tags

- “Everyone knows”  
- “Studies show” (which studies?)  
- “Billions wasted” (itemise)  
- “All migrants / all politicians …” (replace with incentives + distributions)
